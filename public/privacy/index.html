挺足球 — 隱私權政策

生效日期：2026-06-25　|　營運者：挺足球（BAT Mobile）　|　網站：foootball.twtools.cc

1. 本政策涵蓋範圍

本政策說明「挺足球」相關工具與其 Meta 應用程式如何處理資料。此工具僅供本粉專營運者，將自家內容發佈到自己管理的 Facebook 粉絲專頁，不對外提供帳號註冊或一般使用者服務。

2. 我們透過 Meta API 存取哪些資料

本應用程式透過 Facebook 官方 Graph API，使用下列粉專管理權限，且僅作用於營運者本人管理的粉絲專頁：

  • pages_manage_posts：在本粉專建立、排程貼文。
  • pages_manage_engagement：在本粉專貼文下發佈首則留言（放置文章連結）。
  • pages_read_engagement：讀取本粉專基本資訊以確認發佈狀態。

我們不蒐集追蹤者或一般使用者的個人資料，不讀取私訊內容，不建立使用者個人檔案。

3. 資料用途

取得的存取權僅用於單一目的：把營運者自行製作、且已人工審核過的內容（文字與圖卡）發佈或排程到本粉專。不用於廣告投放、不用於分析第三方、不用於任何自動化決策。

4. 資料儲存與安全

用於發佈的存取權杖（access token）儲存於營運者自有、受控的環境中，不寫入公開程式庫、不傳輸給任何第三方服務。本工具為本機執行，不設對外伺服器收集資料。

5. 資料分享

我們不販售、不出租、不交換任何資料。除非法律要求，否則不向第三方揭露。發佈動作本身會將內容傳送至 Meta 平台，受 Meta 隱私政策 規範。

6. 資料保留與刪除

本工具不保留追蹤者個資，故無此類資料可供刪除。與發佈相關的營運紀錄由營運者自行保管。如需確認或要求刪除任何與本應用程式相關的資料，請來信 charlie.chien@gmail.com，我們將於合理時間內處理。

7. 兒童隱私

本工具非針對 13 歲以下兒童，且不會在知情下蒐集其資料。

8. 政策變更

本政策如有更新，將更新本頁頂部之生效日期。

9. 聯絡方式

對本政策有任何疑問，請來信 charlie.chien@gmail.com。
