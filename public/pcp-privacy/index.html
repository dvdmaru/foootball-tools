Publish Control Plane — 隱私權政策

生效日期：2026-06-25　|　營運者：BAT Mobile　|　應用程式：pcp-publish

1. 本政策涵蓋範圍

Publish Control Plane（pcp-publish）是營運者自用的內部發佈工具，用於把營運者自家製作、且已人工審核的內容，發佈或排程到營運者自己管理的社群粉絲專頁。本工具不對外提供帳號註冊，亦不服務一般終端使用者。

2. 我們透過 Meta API 存取哪些資料

本應用程式透過 Facebook 官方 Graph API，使用下列粉專管理權限，且僅作用於營運者本人管理的粉絲專頁：

  • pages_manage_posts：建立、排程粉專貼文。
  • pages_manage_engagement：於貼文下發佈首則留言（放置連結）。
  • pages_read_engagement：讀取粉專基本資訊以確認發佈狀態。

本工具僅處理 organic（自然觸及）發佈，不申請、不使用任何廣告（ads）權限；不蒐集追蹤者或一般使用者個資、不讀取私訊。

3. 資料用途

取得的存取權僅用於把營運者已審核的內容（文字、圖卡）發佈或排程到其自有粉專。不用於廣告投放、不用於建立使用者輪廓、不進行自動化決策。

4. 資料儲存與安全

存取權杖（access token）儲存於營運者自有、受控的本機環境，不寫入公開程式庫、不傳輸給任何第三方。本工具為本機執行（local agent），不設對外伺服器收集資料。

5. 資料分享

我們不販售、不出租、不交換任何資料。除非法律要求，否則不向第三方揭露。發佈動作會將內容傳送至 Meta 平台，受 Meta 隱私政策 規範。

6. 資料保留與刪除

本工具不蒐集第三方個資。與發佈相關之營運紀錄由營運者自行保管。如需確認或要求刪除任何與本應用程式相關之資料，請來信 charlie.chien@gmail.com。

7. 兒童隱私

本工具非針對 13 歲以下兒童，且不會在知情下蒐集其資料。

8. 政策變更

本政策如有更新，將更新本頁頂部之生效日期。

9. 聯絡方式

對本政策有任何疑問，請來信 charlie.chien@gmail.com。
